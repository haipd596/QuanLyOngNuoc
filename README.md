# Quan Ly Ong Nuoc

Dự án đồ án tốt nghiệp: phân tích, thiết kế và xây dựng website quản lý bán vật liệu điện nước cho cửa hàng Ống Nước Việt.

## Cấu trúc
```bash
quanlyongnuoc/
├─ be/   # Backend
└─ fe/   # Frontend (để trống giai đoạn đầu)
```

## Hướng đi
- Backend dùng NestJS + TypeScript.
- Database dùng MySQL.
- ORM dùng Prisma.
- FE sẽ kết nối qua REST API.

## Các mốc chính
1. Hoàn thiện backend nền.
2. Hoàn thiện database và API.
3. Tích hợp Google login, mail, AI kho.
4. Sau đó mới phát triển giao diện FE.
